# hexo-uma-calendar
hexo page floating window plugin
## 使用方式
```
$ npm install hexo-uma-calendar@0.0.2 --save
```
## 插件配置
在hexo根目录下的_config.yml文件中加入
```
umaCale:
enable: true #是否启用
localtion: 'home' #显示的页面
```

在需要插入的地方加入html代码
```html
<uma-card></uma-card>
```

## 效果
大概会在代码处出现一个悬浮的小窗口
